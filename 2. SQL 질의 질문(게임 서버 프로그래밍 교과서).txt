SQL 질의 구문

여러분이 데이터베이스에 엑세스할 때 필수적으로 알아야 할 문법은 질의 입니다.
여러분이 데이터베이스에 액세스하면서 주로 하는 것은 레코드 추가하기, 읽기, 변경(Update), 삭제 입니다.
각각을 CRUD(Create, Read, Update, Delete)라고 합니다.

데이터베이스 질의 구문은 그 자체만을 다루는 도서가 따로 있을만큼 매우 방대한 내용입니다.
하지만 여기서는 기초적인 내용만 다루며, 플레이어 정보를 저장하거나 로딩하는 정도를 넘어서지 않겟습니다.

그러면 새 레코드를 삽입하는 방법을 알아봅시다. 새 레코드를 삽입할 때는 다음과 같이 insert로 시작하는 구문을 사용합니다.

insert into table1 (a,b,c) values (1,2,3)

테이블 table1에 필드 a=1, b=2, c=3 이 들어 있는 새 레코드를 하나 추가하는 내용입니다.
테이블 table1의 모든 레코드를 얻으려면 다음과 같이 select로 시작하는 구문을 쓰십시오.

select a,b,c from table1 where a=1

그러면 table1에 있는 레코드 중 필드a가 1인 레코드를 찾습니다. 찾은 레코드의 필드 a,b,c,를 얻습니다.

select * from table1 where a=1

table1에서 필드 a가 1인 레코드를 찾습니다. 그리고 그것의 모든 필드를 얻습니다. 
테이블 안의 레코드 필드 값을 변경하는 것은 update로 시작하는 구문입니다.

update table1 set b=2 where a=1

table1에 있는 레코드 중 필드 a가 1인 레코드를 찾아 그것의 필드 b를 2로 변경합니다.
이번에는 레코드 삭제를 알아봅시다. delete로 시작하는 구문입니다.

delete from table1 where a=1

이 문장을 실행하면 table1에 있는 레코드 중 필드a가 1인 것을 모두 삭제합니다.
